# Housing Price Prediction

This is a beginner Machine Learning regression project built using Python and Scikit-learn.

## Features Used

* Bedrooms
* Bathrooms
* Sqft Living Area

## Libraries Used

* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## ML Algorithm

* Linear Regression

## What This Project Does

Predicts house prices based on house features.

## Concepts Learned

* Regression
* Train-Test Split
* Model Training
* Prediction
* Mean Absolute Error
* Data Visualization
